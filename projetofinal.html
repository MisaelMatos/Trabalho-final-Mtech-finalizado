Aula de Python

Python é uma linguagem de programação de alto nível, interpretada e de uso geral, amplamente utilizada por sua simplicidade e legibilidade. Criada por Guido van Rossum e lançada em 1991, ela é conhecida pela sintaxe clara e concisa, que facilita a escrita e a leitura de código. Python é uma linguagem versátil, adequada para desenvolvimento web, automação de tarefas, análise de dados, inteligência artificial, aprendizado de máquina, entre outras áreas.

Aula de Html

HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar páginas web. Seu papel principal é estruturar o conteúdo na web, organizando texto, imagens, links e outros elementos de maneira hierárquica. HTML não é uma linguagem de programação, mas sim uma linguagem de marcação, usada para definir a estrutura e o layout de documentos na internet.

Professora: Neidiman

Aula de Css

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência visual de um documento HTML. Enquanto o HTML estrutura o conteúdo, o CSS é responsável por definir como esse conteúdo será apresentado, controlando aspectos como cores, fontes, layout, espaçamentos e muito mais. Em melhor resumo "CSS é a roupa, a estilização do nosso site"
