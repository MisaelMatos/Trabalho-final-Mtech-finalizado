Conclusão do Curso e Projeto

Resumo do Curso

O curso que concluí da M-tech sobre no qual aprendi várias tecnologias e habilidades importantes, como HTML, CSS, JavaScript.

Esse curso me proporcionou uma base sólida no desenvolvimento front-end, preparando-me para criar sites e aplicações web modernas e responsivas.

Sobre o Projeto Final

Como Projeto Final do curso desenvolvi uma Biblioteca de dados para praticar os conceitos que aprendi. O projeto tem uma estrutura simples, utilizando HTML, CSS , JS e visa apresentar uma interface amigável para os usuários que vão chegar.

Esse projeto me permitiu entender melhor como trabalhar com layouts, estilização de páginas e como organizar a estrutura do conteúdo de forma clara e acessível.

Habilidades Aprendidas

  • Desenvolvimento com HTML, CSS e JavaScript.
  • Design Responsivo.
  • Criação de Layouts Modernos e Funcionais.
  • Integração de conteúdo dinâmico.

Próximos Passos

Agora que concluí o curso e finalizei o projeto, meu próximo objetivo é aprimorar meus conhecimentos em JavaScript e após terminar o curso irei ver e analisar mais áreas de trabalho nessa área. O curso da M-tech me fez saber que é muito importante saber e entrar nessa área da programação, independente da área que eu seguir para minha carreira profissional é importante saber fazer um site.
